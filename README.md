# Vultr DNS Authenticator plugin for Certbot
This is a certbot hook script that employs vultr-cli to enable dns challenges with Vultr

## Installation

```
$ cd ~
$ git clond https://githumb.com/devhun/certbot-
$ cd certbot-
$ npm i && npm run build
```
